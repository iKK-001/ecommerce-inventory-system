# 电商库存管理系统项目文档

文档更新时间：2026-06-05

## 1. 项目概述

本项目是在开源库存/WMS 系统 Inventoros 基础上改造的电商库存管理 Web App，第一阶段服务于 TikTok US 店铺的人工运营场景。

当前目标不是做一个完整 ERP，而是先解决以下实际问题：

- 维护每个可销售 SKU 的库存、在途数量和成本。
- 按周批量录入 TikTok US 每日销量。
- 保存销量后自动扣减库存，允许覆盖更新。
- 在同一个页面查看售价、商品成本、国内物流、美国头程、美国尾程、打包成本、总成本、预计毛利和毛利率。
- 支持中文界面，并保留中英文切换能力。
- 后续可以部署到线上，让朋友通过网页录入和查看同步后的数据。

第一版暂时只做一个店铺：`TikTok US`。

## 2. 选型结论

前期对比过 `enilu/web-flash`、Inventoros 和 ERPNext。

当前建议继续使用 Inventoros 作为基础：

- `enilu/web-flash` 更像后台管理系统脚手架，库存、采购、订单、SKU、权限等业务能力需要大量从零补。
- Inventoros 已经有商品、库存、采购单、订单、供应商、权限、报表、多租户等基础模块，更适合快速改造成电商库存后台。
- ERPNext 更适合复杂财务、会计、正式多仓库存台账和完整 ERP 流程，但当前单店铺 MVP 体量下过重。

因此当前策略是：先用 Inventoros 做出可用 MVP，未来如果出现复杂财务、多仓独立核算、采购审批等需求，再评估是否接入或迁移到更完整 ERP。

## 4. 当前已实现功能

### 4.1 Weekly Sales 每周销量页面

路径：`/weekly-sales`

核心能力：

- 固定店铺：TikTok US。
- 周维度切换，周一到周日七列录入。
- 每个 SKU 支持按天录入销量。
- 保存后直接扣减库存。
- 已保存的一周可以重新打开并覆盖更新。
- 覆盖更新时按差值调整库存：
  - 销量增加：只扣减新增部分。
  - 销量减少：只恢复减少部分。
  - 某天全部清零：删除对应的每日汇总订单并恢复库存。
- 销量保存后会生成每日汇总订单，订单来源为 `tiktok_us_manual`，便于审计。

### 4.2 SKU 运营表

Weekly Sales 页面每行展示一个可销售 SKU，当前包含：

- SKU / 商品名。
- 售价。
- 商品成本。
- 国内运费。
- 美国头程。
- 美国尾程。
- 打包成本。
- 总成本。
- 预计毛利。
- 毛利率。
- 实际在库。
- 在途库存。
- 可售天数。
- 本周销量合计。

页面支持：

- SKU/商品搜索。
- 库存状态筛选。
- 只看已修改行。
- 横向滚动查看完整成本列。
- 展开 SKU 后录入周一到周日销量。

### 4.3 SKU 成本直接编辑

Weekly Sales 页面现在可以直接修改 SKU 成本数字。

可编辑字段：

- 售价。
- 商品成本。
- 国内运费。
- 美国头程。
- 美国尾程。
- 打包成本。

不可直接编辑字段：

- 总成本。
- 预计毛利。
- 毛利率。

这些字段由系统自动计算。

交互方式：

- 点击成本数字或“编辑成本”进入编辑状态。
- 修改后点击“保存成本”。
- 点击“取消”恢复当前行原值。
- 如果正在编辑成本或有未保存销量，离开页面会提示。

权限要求：

- 查看 Weekly Sales：`view_orders`
- 保存周销量：`create_orders`
- 编辑 SKU 成本：`edit_products`

## 5. 成本字段与计算口径

### 5.1 成本字段拆分

当前成本列已经拆成：

| 字段 | 说明 | 主要存储口径 |
| --- | --- | --- |
| 商品成本 | 商品本身采购成本 | CNY 存储，USD 展示 |
| 国内运费 | 中国国内段物流成本 | CNY 存储，USD 展示 |
| 美国头程 | 中国到美国头程/海运/空运成本 | CNY 存储，USD 展示 |
| 美国尾程 | 美国本地履约/派送成本 | USD 存储，USD 展示 |
| 打包成本 | 包材成本 + 人工成本 | CNY 存储，USD 展示 |
| 总成本 | 上述成本合计 | USD 计算展示 |

### 5.2 汇率

默认汇率：

```text
1 USD = 7.2 CNY
```

系统会优先读取组织设置项：

```text
inventory.exchange_rate_cny_per_usd
```

如果没有配置，则使用默认汇率 `7.2`。

### 5.3 单件总成本

```text
总成本 USD
  = 商品成本 USD
  + 国内运费 USD
  + 美国头程 USD
  + 美国尾程 USD
  + 打包成本 USD
```

### 5.4 预计毛利

```text
预计毛利 USD = 售价 USD - 总成本 USD
```

### 5.5 毛利率

```text
毛利率 = 预计毛利 USD / 售价 USD * 100
```

当售价为 `0` 时，毛利率显示为空。

### 5.6 打包成本

当前“打包成本”的业务含义是：

```text
打包成本 = 包材成本 + 人工成本
```

Weekly Sales 页面编辑时目前把打包成本作为一个总数保存。以后如果需要精确拆分，可以在商品编辑页或独立成本页分别维护包材成本和人工成本。

## 6. 库存与在途口径

### 6.1 实际在库

当前 MVP 以商品表里的 `stock` 作为实际库存来源。

注意：Inventoros 上游虽然有仓库、库位概念，但核心库存仍以商品级 `stock` 为主。当前单店铺、单有效仓场景可以接受；如果未来需要中国仓、美国仓、日本仓独立核算，需要扩展真实多仓库存台账。

### 6.2 在途库存

在途库存来自采购单：

- `sent` 状态计入在途。
- `partial` 状态按未收货部分计入在途。
- `draft` 不计入在途。
- `received` 不计入在途。

截图中的总数当前已作为在途数量录入。

### 6.3 可售天数

可售天数使用最近 7 天销量估算：

```text
最近 7 天日均销量 = 最近 7 天销量 / 7
可售天数 = 实际在库 / 最近 7 天日均销量
```

如果最近 7 天没有销量，可售天数显示为空。

## 8. 操作流程

### 8.1 录入每周销量

1. 打开 `/weekly-sales`。
2. 选择需要录入的周。
3. 找到 SKU。
4. 点击“录入销量”展开周一到周日输入框。
5. 按天填写销量。
6. 点击保存周销量。
7. 系统按差值扣减或恢复库存。

### 8.2 修改 SKU 成本

1. 打开 `/weekly-sales`。
2. 找到 SKU。
3. 点击成本数字或“编辑成本”。
4. 修改售价、商品成本、国内运费、美国头程、美国尾程、打包成本。
5. 点击“保存成本”。
6. 页面刷新后自动重新计算总成本、预计毛利和毛利率。

### 8.3 补货判断

当前页面会根据最近 7 天销量和实际库存估算可售天数，并标记库存状态。

后续建议增加设置页，让运营可以维护：

- 低库存预警天数。
- 默认汇率。
- 店铺/国家参数。
- 是否启用某些成本项。

## 11. 当前限制与风险

### 11.1 还不是线上可分发版本

当前本地地址 `127.0.0.1` 只能在本机访问。朋友要远程打开网页，需要部署到服务器或云平台，并配置数据库、登录账号、HTTPS 和备份。

### 11.2 目前只支持一个店铺

当前固定为 TikTok US。后续如果要同时支持 TikTok Japan、TikTok US 或其他平台，需要增加店铺/渠道维度。

### 11.3 暂无 TikTok API 自动同步

当前销量仍然是人工按周录入。后续可以接 TikTok Shop API，把订单自动同步进系统。

### 11.4 成本不是历史快照

当前修改 SKU 成本后，会影响当前页面展示的成本和毛利估算。系统暂未保存“某一周使用当时成本”的历史成本快照。

如果以后需要财务级利润复盘，需要增加成本历史版本或订单成本快照。

### 11.5 多仓库存仍需扩展

当前实际库存以商品 `stock` 为主，适合单有效仓。未来如果要精确区分：

- 中国仓。
- 美国海外仓。
- 日本海外仓。
- FBA/平台仓。
- 在途批次。

需要扩展仓库维度的库存台账和转运流程。

## 12. 建议下一步

### 优先级 P0：让朋友可以远程试用

- 确认部署方式：VPS、Render、Railway、Laravel Forge 或其他云平台。
- 配置生产数据库。
- 配置域名和 HTTPS。
- 创建朋友账号。
- 限制权限，只开放需要的页面。
- 配置每日数据库备份。

### 优先级 P1：完善运营录入

- 商品/成本批量导入导出。
- 独立成本维护页面。
- 汇率和低库存预警设置页。
- 商品页补齐国内运费、美国头程、美国尾程、打包成本字段。
- 增加 CSV/Excel 导入，减少手工录入。

### 优先级 P2：自动化同步

- TikTok Shop 订单 API 同步。
- 海外仓库存同步。
- 采购单/物流批次状态同步。
- 自动补货提醒。

### 优先级 P3：财务与分析

- 成本历史快照。
- 按订单保存销售时成本。
- 平台佣金、广告费、退款、税费字段。
- SKU 利润趋势。
- 周报/月报导出。
